# GTA VI Landing Page UI Clone 🌴🚁

A high-performance, cinematic UI clone of the official Grand Theft Auto VI website. This project was built to test handling complex, visual-heavy layouts, custom typography, and high-end scroll interactions within a modern component architecture.

### 💡 Project Notes

*   **The Architecture:** Transitioning into the React ecosystem, this project focuses heavily on clean, modular component design. Elements like the Jason and Lucia character cards and the Leonida Keys exploration section are broken down into reusable structures.
*   **Styling & Animation:** Tailwind CSS was utilized for rapid, pixel-perfect UI matching, while GSAP was integrated into the React environment to handle the smooth, high-end scroll transitions.
*   **Disclaimer:** This is an unofficial, fan-made UI clone built strictly for educational and portfolio purposes. All original designs, assets, and IPs belong to Rockstar Games. 

### 🛠️ Running Locally

To run this project on your local machine:

1. Clone the repository:
   git clone https://github.com/ashishbtech/gta-vi-react-landing.git

2. Navigate to the project directory:
   cd gta-vi-react-landing

3. Install dependencies:
   npm install

4. Start the Vite development server:
   npm run dev
